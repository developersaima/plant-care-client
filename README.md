# 🌿 PlantCare

## 📖 Description

PlantCare is a modern full-stack web application built with **Next.js**, **TypeScript**, and **Tailwind CSS**. The application allows users to explore different plants, learn essential plant care information, and manage their own plant collection through a clean, responsive, and user-friendly interface.

---

## 🎯 Project Purpose

The main purpose of PlantCare is to simplify plant management by providing an easy-to-use platform where users can discover plants, organize their personal collection, and access detailed care information. The project was developed as part of the **Programming Hero SCIC Assignment** to demonstrate modern full-stack web development using TypeScript.

---

## ✨ Key Features

- Responsive and modern user interface
- User authentication
- Explore plants with search functionality
- Filter and sort plants
- View detailed plant information
- Add new plants
- Manage personal plant collection
- Protected dashboard routes
- Form validation
- Clean and reusable component-based architecture

## 🔄 Workflow

- Users can register and log into their accounts.
- Authenticated users can add new plants to the system.
- Users can browse and explore available plants.
- Plants can be searched, filtered, and sorted for better navigation.
- Detailed information is available for every plant.
- Users can manage their own plant collection from the dashboard.
- Protected routes ensure that only authenticated users can access management features.
